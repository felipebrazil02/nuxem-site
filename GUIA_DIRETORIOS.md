## 3. InfoSimples - Cadastro CNPJ (Nota)

### URL
- **Site:** https://infosimples.com

### Esclarecimento
A InfoSimples **NÃO é um diretório empresarial** onde se cadastra a empresa. É uma empresa de **APIs de consulta e OCR** que automatiza consultas em portais públicos (Receita Federal, Sintegra, Sefaz, etc.) para validação de dados cadastrais.

### O Que Fazer na Prática
Para gerar **backlink com CNPJ**, o caminho correto é garantir que a Nuxem esteja com dados atualizados nos **portais públicos oficiais** que alimentam plataformas como a InfoSimples:

1. **Receita Federal** — mantenha o CNPJ ativo e dados atualizados
   - https://www.gov.br/receitafederal
2. **Sintegra / Sefaz-SP** — certifique-se de que a inscrição estadual está regular
   - https://www.sintegra.fazenda.gov.br
3. **Portal Gov.br** — mantenha cadastro atualizado
   - https://www.gov.br

### Recomendação
Se quiser usar os serviços da InfoSimples (pagos), contrate a API de **consulta de CNPJ** para automatizar validações de fornecedores/parceiros da Nuxem. Mas não há "cadastro gratuito" que gere backlink.
